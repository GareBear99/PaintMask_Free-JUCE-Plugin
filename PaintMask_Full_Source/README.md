# PaintMask Studio — DARPA v0.8 Round-Trip Handoff

PaintMask is a visual composition plugin where painting becomes notes, and notes can be deterministically projected back into paint structures.

- Lite fallback

## What v0.8 adds
- MIDI → Paint importer scaffold
- deterministic round-trip mapping spec
- canonical lane/group import rules
- example round-trip JSON mapping
- import/export architecture notes
- updated delivery and truth-state docs
